# Finance Tracker

A simple finance tracker web app for managing your expenses, income, and budgets. Track your financial activity with a clean and easy-to-use interface. Built with Next.js 16, React 19, TypeScript, and Tailwind CSS.

## Run Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/itsdevdeepak/finance-tracker.git
   cd dictionary-web
   ```

2. **Install dependencies:**

   ```bash
   pnpm install
   ```

3. **Set up environment variables:**

   Rename the `example.env` file to `.env.local` in the root directory:

   ```bash
   mv example.env .env.local
   ```

4. **Start the development server:**

   ```bash
   pnpm dev
   ```

5. **Open your browser:**

   Navigate to [http://localhost:3000](http://localhost:3000) to see the app.
